Python is amazing!

But well...

It's not a great choice for heavy computation.

What's heavy computation?

Typically anything that's CPU intensive.

Including (but not limited to):

  • Processing large files
  • Hard math (matrix inversion, prime numbers, etc.)
  • Cryptography (hashing, encryption, etc.)

What does Rust offer?

  • Systems programming language
  • Performance without compromising safety
  • No Garbage Collector!
  • Zero-cost abstractions
  • Fearless concurrency
  • Strong static typing, generics, type inference, mutability, pattern matching, ...

... and a lot more!

Why Rust?

To avoid:

  • Dangling pointers
  • Use after free and double free
  • Out of bounds indexing
  • Iterator invalidation
  • Null pointer dereference
  • Data races in parallelization
